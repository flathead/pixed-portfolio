Кафе платило агрегатору 30% с каждого заказа. Владелец посчитал — за
год это машина. Позвонил мне.

За 6 недель: Telegram-бот с полным циклом — меню, корзина, оплата,
уведомления. Кухня получает заказы в Telegram-канал. Курьеры шлют
геолокацию — клиент видит точку на карте.

## Проблема

Агрегаторы берут 25–35% комиссии. При среднем чеке 800 ₽ и 50 заказах
в день — это **10 000 ₽ в день** или ~300 000 ₽ в месяц. За год —
машина.

## Решение

Telegram-бот с полным циклом доставки. Без приложения, без сайта —
только мессенджер, который уже есть у всех.

## Техническая часть

```
Стек: PHP + ReactPHP (async) + Telegram Bot API
База: MySQL
Деплой: VPS + Nginx + Supervisor
```

Самое сложное — **WebSocket-сервер для реального времени**. Написал
кастомный async-демон на ReactPHP. Курьеры отправляют координаты →
клиент видит точку на карте.

### Интеграции

- **Оплата:** ЮКасса inline
- **Кухня:** отдельный Telegram-канал с кнопками
- **Аналитика:** простой дашборд для владельца

## Результат

- Запуск через 6 недель
- Окупился через 2 месяца
- Работает без сбоев 2+ года
